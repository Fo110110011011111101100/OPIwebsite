Разработка структуры программы

– Класс User предназначен для работы с учетными записями пользователей и администраторов. Содержит четыре свойства и два метода.

– Класс Admin предназначен для реализации функционала доступного администратору. Содержит сем свойств и одинадцять методов.

– Класс Staff предназначен для работы с водителями таксопарка. Содержит восемь свойств и двадцать один метод.

– Класс Database предназначен для работы с данными водителей. Содержит два свойства и двадцать четыре метода.
